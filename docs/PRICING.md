# Pricing Model

Plans:
- **Solo** — 1 seat, base words
- **Team** — base seats + word multiplier, add seats, volume discounts

Billing:
- Stripe Checkout & Customer Portal
- Seats managed on team
- Usage chart and invoices in `/settings/billing`

Multipliers (defaults):
- `gpt-4o: 1.0`
- `claude-3-7-sonnet: 0.7`
- `gemini-pro: 0.1`
